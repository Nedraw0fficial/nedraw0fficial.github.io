TECH STACK

Built the whole thing from scratch with modern web tech but kept that retro aesthetic. The site uses HTML5 for structure, CSS3 for all the styling and animations, and plain JavaScript for the interactive stuff. The Space Invaders game runs on HTML5 Canvas - didn't need any game engines or anything.

  • HTML5 - Page structure and Canvas for the game
  • CSS3 - All the animations, neon effects, scanlines, and responsive layout
  • JavaScript - Animations and the game logic
  • Google Fonts - Press Start 2P and VT323 for that retro font look
  • Responsive Design - Works on anything, even on Samsung refrigirators
