TECH STACK

This project showcases modern web development skills while maintaining that classic retro aesthetic. The site uses semantic HTML5, advanced CSS3 with animations and effects, and vanilla JavaScript for interactivity. The Space Invaders game is built entirely with HTML5 Canvas - no game engines needed!

  • HTML5 - Semantic structure and Canvas for game rendering
  • CSS3 - Custom animations, neon effects, scanlines, and responsive design
  • JavaScript - Interactive animations and full game logic
  • Google Fonts - Press Start 2P and VT323 for authentic retro typography
  • Responsive Design - Works great on desktop, tablet, and mobile
